Required For Case:
- x1 ALpha28 PCB
- x28 Through-Hold Diodes
- x1 Pro-Micro or Pro-Micro Clone 
- x4 Rubber Feet
- x1 Alpha28 "Half-Plate" PCB Switch Plate
- x1 Alpha28 PCB Bottom Base Plate
- x4 9mm+ M2 Spacers
- x8 M2 Screws
